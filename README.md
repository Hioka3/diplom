# Корпоративный новостной портал

Веб-приложение для внутреннего корпоративного новостного портала: новости, объявления, категории, поиск, фильтрация, авторизация и разграничение ролей.

## Стек

- Python + Flask
- MySQL
- HTML
- CSS
- JavaScript

## Возможности

- Автоматическое создание базы данных и таблиц при первом запуске.
- Авторизация по логину и паролю.
- Роли пользователей: `admin`, `editor`, `employee`.
- Новостная лента в обратном хронологическом порядке.
- Создание, редактирование и удаление публикаций.
- Категории новостей.
- Поиск по заголовку, краткому описанию и тексту.
- Фильтрация по категории и датам.
- Загрузка изображения или файла к публикации.
- Адаптивный современный интерфейс.

## Установка

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Убедитесь, что MySQL запущен.

3. При необходимости создайте файл `.env` рядом с `app.py`:

```env
SECRET_KEY=your-secret-key
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=your_mysql_password
MYSQL_DATABASE=corporate_news_portal
UPLOAD_FOLDER=uploads
```

4. Запустите приложение:

```bash
python app.py
```

При первом запуске приложение само создаст базу данных, таблицы, категории, тестовых пользователей и первую новость.
